# @brc-analytics/ga2

Genome Ark 2: explore and analyze reference genome assemblies using Galaxy workflows.

The Genome Ark 2 Next.js app. Site-agnostic code is imported from `@brc-analytics/core`; everything here is GA2-specific.

## Contents

- `pages/` — GA2 routes (organisms, assemblies, workflows, analyze, about, roadmap)
- Site-specific views (`GA2HomeView`, GA2 About/Roadmap/Partner-Resources)
- Site-specific components (`OrganismAvatar`, GA2 home sections, GA2 `Branding`)
- GA2 concrete entity types, column/category configs, theme overrides
- GA2 catalog source/output + build scripts, `public/` assets, `next.config`, `.env`
